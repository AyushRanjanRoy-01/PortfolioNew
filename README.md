# Ayush Ranjan Roy — Portfolio

Clean professional portfolio for **[ayushranjanroy.com](https://ayushranjanroy.com)**.

Design inspired by top AI engineer / researcher sites (Karpathy-style clarity, swyx density, modern type hierarchy) — not a template clone.

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Vercel (`web/` is the deploy root)

## Develop

```bash
cd web
npm install
npm run dev
```

## Content

Edit `web/lib/content.js` — single source of truth for projects, experience, about, and links.

## Deploy

Push to `main`. Vercel builds from the `web` directory.
